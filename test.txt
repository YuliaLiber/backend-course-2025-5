це файл з mac

1. Перевірка роботи сервера без коду
curl http://127.0.0.1:3000/

2. GET — отримати картинку для коду 200 і зберегти у файл 200.jpg
curl -v http://127.0.0.1:3000/200 --output 200.jpg

3. PUT — зберегти (створити або замінити) файл у кеші з кодом 555
curl -v -X PUT --data-binary "@test.txt" http://127.0.0.1:3000/555

4. GET — перевірити, що файл 555 існує і його можна прочитати
curl -v http://127.0.0.1:3000/555

5. DELETE — видалити файл 555 з кешу
curl -v -X DELETE http://127.0.0.1:3000/555

6. GET — перевірити, що файл 555 видалено (очікується 404)
curl -v http://127.0.0.1:3000/555
